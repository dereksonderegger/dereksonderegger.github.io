Stat 570L - Introduction to R

Introduction to R

This a 1-credit course intended to be a companion to the Stat 570 course. We will cover basic data types and structures, probability distributions, flow control, creation of functions, importing data, graphing, and standard statistical analyses.

This course will not cover any statistical theory, but will give the students a way to apply what they are learning in Stat 570 in software.

I’ve not yet found a book on R that is suitable for the course, so I’ve devoloped my own. The notes I’ve developed for this class are available through my GitHub site and I strongly encourage people to submit error reports and expand upon these notes.

Books:

R for Everyone by Jared Lander is a book very similar in spirit to the notes I prepared (I wish I found the book before I wrote my own notes!), but are more polished. 

The R Graphics Cookbook and associated website by Winston Chang are great for learning R and how to create great graphics using ggplot2.

The R Book by Michael Crawly is more oriented towards documenting statistical procedures.
