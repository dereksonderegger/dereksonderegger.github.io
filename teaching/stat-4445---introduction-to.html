Stat 444/5 - Introduction to R

This course sequences is intended to introduce R to students in both the BS Data Science degree program and to users of R in any other university course. 

STA 444 is a 1-credit course intended to be a gentle introduction to R and covers enough content to be able to do simple data summarization, graphing, and model interpretation as well as produce reasonable report style output documents.

STA 445 is a 2-credit course intended to dive into data wrangling in more technical detail and by the end of the course, students will be much more comfort with coding within the R tidyverse.

The notes I’ve developed for this class are available through my GitHub site and I strongly encourage people to submit error reports and expand upon these notes.
